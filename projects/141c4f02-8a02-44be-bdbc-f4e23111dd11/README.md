# BMI Calculator

A comprehensive Body Mass Index (BMI) calculator application with support for multiple units and classification standards.

## Features

- **BMI Calculation**: Calculate BMI using the standard formula: BMI = weight(kg) / height(m)^2
- **Multiple Unit Systems**: Support for both metric (kg/cm/m) and imperial (pounds/feet-inches) units
- **Classification Standards**: Choose between WHO and Asian classification standards
- **History Tracking**: Save and view your BMI calculation history using local SQLite database
- **Interactive Interface**: Easy-to-use command-line menu system
