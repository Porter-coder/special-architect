# Simple Calculator Application

A fully functional calculator application built with Python and tkinter.

## Features

- **Basic Arithmetic**: Addition, subtraction, multiplication, division
- **Parentheses Support**: Use ( and ) to change operation order
- **Percentage**: Convert numbers to percentages with %
- **Square Root**: Calculate square roots with √
- **Sign Toggle**: Switch between positive and negative with +/-
- **History**: View your recent calculations
- **Clear Functions**: C to clear all, ← to delete last character

## Installation

No external dependencies required. The application uses only Python standard libraries.
