# Python Calculator

A comprehensive command-line calculator application written in Python that provides basic arithmetic, scientific calculations, base conversion, and statistical functions.

## Features

### Basic Arithmetic Operations
- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)
- Integer Division (//)
- Modulo (%)

### Scientific Calculations
- Power (exponentiation)
- Square Root
- Factorial
- Absolute Value
- Trigonometric Functions (Sine, Cosine, Tangent)
- Support for both Degree and Radian modes
- Logarithm (base 10)
- Natural Logarithm (ln)
- Logarithm with custom base

### Base Conversion
- Decimal to Binary
- Decimal to Octal
- Decimal to Hexadecimal
- Binary to Decimal
- Octal to Decimal
- Hexadecimal to Decimal

### Statistical Calculations
- Arithmetic Mean
- Geometric Mean
- Median
- Mode
- Standard Deviation
- Variance

## Installation

No external dependencies required. The calculator uses only Python standard library modules.
