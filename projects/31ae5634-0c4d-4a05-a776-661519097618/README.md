# Simple Calculator

A feature-rich command-line calculator written in Python.

## Features

- **Basic Arithmetic**: Addition, Subtraction, Multiplication, Division
- **Advanced Operations**: Power, Square Root, Modulo, Percentage
- **Memory Functions**: M+, M-, MR, MC
- **Edit Functions**: Undo, Redo, All Clear (AC), Delete Last (DEL)

## Installation

No additional dependencies required. This calculator uses only Python standard library modules.
