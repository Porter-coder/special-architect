# Simple Calculator Application

A versatile calculator application built in Python that supports both graphical (GUI) and command-line (CLI) interfaces.

## Features

- **Basic Arithmetic Operations**: Addition (+), Subtraction (-), Multiplication (*), Division (/)
- **Extended Operations**: Modulo (%), Power (^), Square Root (sqrt()), Percentage (%)
- **Expression Evaluation**: Calculate complex expressions like `2+3*4` with proper operator precedence
- **Parentheses Support**: Use `()` to change calculation order
- **History Tracking**: Keep track of your recent calculations
- **Dual Interfaces**: Choose between GUI and CLI modes
