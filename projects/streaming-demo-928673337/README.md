# Todo List Manager

A simple command-line todo list application to help you manage your daily tasks and track your productivity.

## Features

- **Task Management**: Create, read, update, and delete tasks
- **Priority Levels**: Assign Low, Medium, or High priority to tasks
- **Categories**: Organize tasks by categories (Work, Personal, Learning, etc.)
- **Due Dates**: Set deadlines for your tasks
- **Completion Tracking**: Mark tasks as complete or incomplete
- **Search**: Find tasks by keyword in title or description
- **Filtering**: Filter tasks by status or category
- **Sorting**: Sort tasks by creation date, priority, due date, or title
- **Statistics**: View task statistics and progress
- **Auto-Save**: Tasks are automatically saved to a JSON file
