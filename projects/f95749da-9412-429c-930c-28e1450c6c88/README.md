# Timer Application

A multi-function timer software built with Python and tkinter.

## Features

- **Stopwatch Mode**: Track elapsed time from zero
- **Countdown Mode**: Set custom duration timers with auto-stop
- **Multiple Timers**: Create and manage several timers simultaneously
- **History Tracking**: Automatically save and review past timer sessions
- **Precise Timing**: Millisecond-level accuracy for accurate time tracking

## Installation

1. Ensure you have Python 3.6 or higher installed
2. No additional dependencies required - uses only Python standard library
