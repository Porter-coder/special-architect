# TODO List Application

A simple yet powerful command-line TODO list application to help you manage your tasks efficiently.

## Features

- **Task Management**: Create, view, edit, and delete tasks
- **Priority Levels**: Assign priority (low, medium, high) to tasks
- **Status Tracking**: Track task status (todo, in-progress, completed, archived)
- **Categories**: Organize tasks by custom categories
- **Due Dates**: Set due dates for tasks
- **Search**: Search tasks by keyword
- **Statistics**: View task completion statistics
- **Data Persistence**: Tasks are automatically saved to a JSON file

## Installation

No installation required. This application uses only Python standard library modules.
