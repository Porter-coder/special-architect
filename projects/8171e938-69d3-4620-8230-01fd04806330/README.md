# Simple Calculator Application

A fully functional calculator application built with Python and tkinter.

## Features

- **Basic Arithmetic Operations**: Addition, subtraction, multiplication, and division
- **Extended Operations**: Percentage (%), square (x^2), square root (sqrt), and power (^)
- **Parentheses Support**: Control operation precedence with ( and )
- **Memory Functions**: Memory Clear (MC), Memory Recall (MR), Memory Add (M+)
- **Expression History**: View and track your recent calculations
- **Continuous Calculation**: Use results in subsequent calculations

## Requirements

- Python 3.6 or higher
- tkinter (included with standard Python installation)

No external dependencies required.

## Installation

1. Clone or download the calculator application files
2. Ensure you have Python installed on your system
