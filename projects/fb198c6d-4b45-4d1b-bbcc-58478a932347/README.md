# Todo List Application

A simple command-line based todo list application for managing your daily tasks.

## Features

- **Create Tasks**: Add tasks with title, description, priority, due date, and tags
- **View Tasks**: Display all tasks in a clean list format
- **Complete Tasks**: Mark tasks as completed with visual indicators
- **Delete Tasks**: Remove unwanted tasks
- **Update Tasks**: Modify any task properties
- **Search Tasks**: Find tasks by keyword in title or description
- **Filter Tasks**: Filter by completion status (all/completed/incomplete)
- **Sort Tasks**: Sort by ID, priority, due date, or creation date
- **Persistent Storage**: Tasks are saved to a JSON file automatically
