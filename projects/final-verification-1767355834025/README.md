# Simple Calculator

A basic calculator application supporting addition, subtraction, multiplication, and division operations with history tracking.

## Features

- **Basic Operations**: Addition (+), Subtraction (-), Multiplication (*), Division (/)
- **Decimal Support**: Precise calculations with floating-point numbers
- **Negative Numbers**: Full support for negative number operations
- **History Tracking**: Stores the last 10 calculation results
- **Continuous Calculations**: Use the previous result in the next calculation
- **Cross-Platform**: Works on Windows, macOS, and Linux

## Installation

This application uses only Python standard library modules, so no external dependencies are required.
