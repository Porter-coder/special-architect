# Todo List Manager

A simple and efficient command-line task management application built in Python.

## Features

- **Task Management**: Create, update, delete, and organize tasks
- **Categories**: Organize tasks into custom categories
- **Tags**: Add flexible tags to tasks for better organization
- **Priority Levels**: Low, Medium, High, Urgent
- **Status Tracking**: Pending, In Progress, Completed, Archived
- **Deadline Reminders**: Automatic overdue task notifications
- **Bulk Operations**: Update or delete multiple tasks at once
- **Statistics**: View your task completion progress
- **Data Persistence**: All data saved to JSON file

## Requirements

- Python 3.6 or higher
- No external dependencies required (uses standard library only)
