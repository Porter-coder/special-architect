# Todo List Application

A simple command-line todo list manager built with Python.

## Features

- **Task Management**: Create, view, edit, and delete tasks
- **Task Details**: Add title, description, priority, due date, and category
- **Priority Levels**: High, Medium, and Low priority options
- **Due Date Tracking**: Set deadlines and see overdue tasks
- **Categories**: Organize tasks by project or category
- **Completion Tracking**: Mark tasks as completed/pending
- **Filtering & Sorting**: Filter by status, category, priority; sort by various criteria
- **Statistics**: View task counts and status overview
- **Data Persistence**: Tasks saved automatically to `tasks.json`

## Requirements

- Python 3.7 or higher
- No external dependencies (uses only Python standard library)
