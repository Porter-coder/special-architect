# Weather Forecast System

A comprehensive weather monitoring and forecasting application built with Python and Tkinter.

## Features

- **Current Weather**: Real-time temperature, humidity, pressure, wind, visibility, and UV index
- **Hourly Forecast**: 24-hour weather predictions with precipitation probability
- **7-Day Forecast**: Daily high/low temperatures and weather conditions
- **Air Quality Monitoring**: AQI gauge and pollutant levels (PM2.5, PM10, NO₂, O₃, CO, SO₂)
- **Weather Alerts**: Active weather warnings with safety recommendations
- **Data Visualization**: Temperature, precipitation, humidity, and wind charts
- **Multi-City Support**: Quick selection from 10 preset cities worldwide
